## Requirements

- [FVM](https://fvm.app/)
- Platform SDKs: Android and XCode
- Registred Android and iOS projects in Firebase

## Setup

```bash
# Get dependencies
fvm flutter pub get

# Download Firebase configuration
fvm dart run flutter_toolkit:cli project-setup
```

## Run

```bash
fvm dart run flutter_toolkit:cli run-app
```

## Unit tests

```bash
fvm flutter test
```
